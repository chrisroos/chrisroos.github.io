## Buffers

- `ls` - List open buffers
- `bp` - Previous buffer

## Windows

- `vs` - Split window vertically
- `ctrl+w` - Switch window

## RSpec

- `! rspec %` - Run rspec for the open file

## Folding

- `:set foldmethod=indent` - Fold based on indent
- `zo` - Open fold
- `zc` - Close fold
- `zM` - Close all folds
- `zR` - Open all folds
